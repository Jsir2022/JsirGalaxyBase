## 这个项目现在是什么

`CustomMod` 当前是一个面向 GTNH 服务器的自定义制度模组。

它的核心目标包括：

- 职业
- 市场与经济
- 贡献度 / 声望
- 公共订单 / 公共工程
- 群组服同步核心状态
- 后续玩法能力扩展

其中，像连锁挖掘这样的功能，会作为独立能力模块接入，而不是直接把制度核心搅乱。

## 这个项目现在不是什么

- 不是普通客户端整合包说明文档
- 不是 Bukkit 插件项目
- 不是多仓库拆分项目
- 不是已经完成的正式玩法发布包

当前更准确的理解是：

- 一个还在持续设计和落地中的 GTNH 自定义模组仓库
