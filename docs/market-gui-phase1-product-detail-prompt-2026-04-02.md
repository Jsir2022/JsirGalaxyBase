# 市场 GUI 第一轮旧 Prompt 已废弃

这份 prompt 对应的是一条已经撤销的旧方向：把市场后续实现继续收窄成单一的标准商品撮合页，并且一度错误滑向了单一材料专场语境。

当前它不再是后续实现依据。

## 当前正式方向

市场正式拆成三类：

1. 标准商品市场
2. 定制商品市场
3. 汇率市场

其中：

- 标准商品市场不再默认等同于金属专场
- 定制商品市场才是玩家间的跳蚤市场 / 二手交易平台
- 汇率市场单独处理任务书硬币、星光币和其他正式货币之间的兑换关系

## 如何使用这份旧文件

- 只把它当作一次已废弃的历史草案记录
- 不要继续按这份 prompt 推进实现
- 后续市场阶段应先补三市场总设计，再分别推进三条市场线
