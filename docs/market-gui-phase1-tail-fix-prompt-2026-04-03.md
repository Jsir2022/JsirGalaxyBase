# 市场第一阶段旧收尾 Prompt 已废弃

这份 prompt 对应的是已经撤销的旧方向：把市场第一阶段收口继续建立在单一标准商品撮合方案上，并且一度错误滑向单一材料专场语境。

当前它不再是后续实现依据。

## 当前正式方向

市场正式拆成三类：

1. 标准商品市场
2. 定制商品市场
3. 汇率市场

其中：

- 标准商品市场负责标准商品目录、统一托管、订单簿撮合和结算
- 定制商品市场负责玩家间非标商品、二手商品、单件商品和跳蚤市场式交易
- 汇率市场负责任务书硬币、星光币和其他正式货币之间的兑换关系

## 后续不再沿用的旧前提

- 不再把标准商品市场默认写成金属专场
- 不再把旧单一路线当成整个市场系统总方向
- 不再让后续实现继续围绕“金属市场补尾巴”展开

## 后续应该怎么做

- 先补三市场总设计和边界文档
- 再分别推进标准商品市场、定制商品市场和汇率市场
- 所有新的实现 prompt 都应以三市场结构为前提
